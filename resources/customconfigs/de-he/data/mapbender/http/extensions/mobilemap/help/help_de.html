Ebenensteuerung:
Als Hintergrundkarten (Baselayer) kann jeweils immer nur eine aktiviert werden. Tapen Sie dazu auf den Optionsschalter oder den Namen der Hintergrundkarte.
Die weiteren Ebenen (Overlays) können beliebig dazugeschaltet werden. Aktivieren bzw. deaktivieren Sie dazu die jeweilige Checkbox.
Um eine Ebene abzufragen (Query) aktivieren Sie den Info- Optionsschalter und tapen danach in der Karte auf das Objekt der Ebene. Es kann immer nur eine Ebene gleichzeitig für die Abfrage aktiviert werden.
Gesten für Touchgeräte:
Pinch-Zoom: Hinein- bzw. Hinauszoomen durch das gleichzeitige Zusammen- bzw. Auseinanderfahren von zwei Fingern über der Karte.(Hinweis: Unter Android 2.x funktioniert diese Geste noch nicht!)
Double Tap - Zoom: Durch das schnelle doppelte Tappen mit einem Finger in die Karte zoomen Sie in den entsprechenden Bereich herein.
Pan: Durch das "Festhalten" der Karte nach einem einfachen Tap und das gleichzeitige Bewegen können Sie den Kartenausschnitt beliebig verschieben.
Einfacher Tap: Nach einem einfachen Tap können Sie eine Abfrage auslösen (vgl. Info-Werkzeug)
